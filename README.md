## 📝 Getting Started (Local Development)

1. **Clone the Repository:**
   ```bash
   git clone <your-repo-url>
   cd netflix_portfolio
   ```
2. **Install Dependencies:**
   ```bash
   npm install
   ```
3. **Run the Project:**
   ```bash
   npm start
   ```
4. **Visit:**
   Open your browser to `http://localhost:3000`
